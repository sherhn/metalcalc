Онлайн калькулятор устойчивости центрально-сжатых стальных стержней

Калькулятор выполняет проверку устойчивости центрально-сжатых стальных стержней (колонн, стоек, раскосов) согласно СП 16.13330.2017 «Стальные конструкции». Расчёт ведётся по условию N / (φ · A · Ry · γc) ≤ 1.

Коэффициент продольного изгиба φ – кривые устойчивости a, b, c

Коэффициент φ определяется по типу поперечного сечения (кривые a, b, c из приложения Д СП 16) и условной гибкости λ̄ = λ · √(Ry/E). Инструмент автоматически интерполирует таблицу значений и отображает промежуточные результаты в реальном времени.

Поддерживаемые марки стали

C235, C245, C255, C345, C345K, C355, C355-1/K, C390/C390-1, C440, C590, C690. Расчётное сопротивление Ry подбирается автоматически по марке стали и толщине проката из таблицы В.5 СП 16.13330.2017.
